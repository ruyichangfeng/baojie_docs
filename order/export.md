# 导出接口
### 说明

| Key      | Value                       |
|:---------|:----------------------------|
| 功能描述 | 保洁单导出          |
| 接口地址 | /baojie/order/export |
| 调用方式 | GET、POST                       |

### 参数

| 参数 | 类型    | 描述 | 说明            | 可空 |
|:-----|:--------|:-----|:----------------|:-----|
| pagesize | integer | 导出页数 |  | N    |


### 接口返回值
无返回值，直接Excel
