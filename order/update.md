# 更新接口
### 说明

| Key      | Value                       |
|:---------|:----------------------------|
| 功能描述 | 保洁单更新          |
| 接口地址 | /baojie/order/update |
| 调用方式 | POST                       |

### 参数

| 参数 | 类型    | 描述 | 说明            | 可空 |
|:-----|:--------|:-----|:----------------|:-----|
| id | integer | 保洁单ID |  | N    |
| checkImgs | string | 验收图片 |验收必须  | N    |
| cleanImgs | string | 保洁图片 | 保洁完成必须 | N    |
### 接口返回值
```json
{
    //返回状态码,非0时为错误
    "resultCode": 0,
    "resultMsg": "操作成功",
    "result": {
        "id": "保洁单ID"
    }
}
```
