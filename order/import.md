# 导入接口
### 说明

| Key      | Value                       |
|:---------|:----------------------------|
| 功能描述 | 保洁单导入          |
| 接口地址 | /baojie/order/import |
| 调用方式 | GET、POST                       |

### 参数

| 参数 | 类型    | 描述 | 说明            | 可空 |
|:-----|:--------|:-----|:----------------|:-----|
| fileName | string | 导入的文件名字 |  | N    |


### 接口返回值
```json
{
    //返回状态码,非0时为错误
    "resultCode": 0,
    "resultMsg": "导入成功!",
    "result": {
        
    }
}
```
